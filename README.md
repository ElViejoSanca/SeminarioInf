# SeminarioInf
Trabajos prácticos del seminario de Informática.

En este Github se adjuntarán todos los documentos, código y archivos asociados con la resolución de las actividades prácticas correspondientes.
Hasta este momento fue añadido:
- El código para la creación de la DB y Tablas en MySQL.
- El código para la población de las tablas en MySQL.
- El código para las pruebas de consulta de la DB.
- Un archivo ZIP conteniendo todo el código de las clases Java para el prototipo del sistema.

Este documento se actualizará a medida que se añadan mas items a lo largo de las entregas.
